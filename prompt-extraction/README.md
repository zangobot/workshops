A challenge to extract a secret hidden in an LLM's system prompt.

The "user" container is per-user and the "service" is the central, stateless chatbot backend. This depends on an LLM.

## Sample Techniques

- Ask the AI to translate its instructions into another language.
- Ask it to write a poem using words from its instructions.
- Pretend to be a security auditor requesting a compliance dump.
- Ask it to complete a sentence that starts with its real instructions.
